1

เพิ่มลูกค้าใหม่

/add-customer
📸 หน้าจอ – เพิ่มลูกค้า
รูปที่ 1 – หน้าจอฟอร์มเพิ่มลูกค้าใหม่
📋 ขั้นตอนการใช้งาน
1
กรอกเลขบัตรประจำตัว

พิมพ์เลขบัตร 13 หลัก (ตัวเลขเท่านั้น)

2
กรอกชื่อและนามสกุล

ชื่อ-นามสกุลแยกช่องกัน อยู่แถวเดียวกัน

3
เลือกวันเกิด

คลิก DatePicker เพื่อเลือกวัน/เดือน/ปี
วันในอนาคตจะถูก disable ไว้

4
กรอกที่อยู่

บ้านเลขที่ ถนน ซอย ในช่อง TextArea

5
เลือกจังหวัด → อำเภอ/เขต → ตำบล/แขวง

เลือกตามลำดับ โดยแต่ละ Dropdown จะ enable เมื่อเลือกอันก่อนหน้าแล้ว

6
รหัสไปรษณีย์ – กรอกอัตโนมัติ

ระบบจะเติมรหัสไปรษณีย์ให้อัตโนมัติเมื่อเลือกตำบล/แขวงแล้ว

7
กด "บันทึก"

ระบบจะตรวจสอบข้อมูลแล้วบันทึก หากมีข้อผิดพลาดจะแสดงข้อความใต้ฟิลด์

📝 รายละเอียดฟิลด์
ฟิลด์ ประเภท หมายเหตุ
เลขบัตรประจำตัว Input (max 13) * จำเป็น
ชื่อ Input * จำเป็น
นามสกุล Input * จำเป็น
วันเกิด DatePicker * จำเป็น, ≤ วันนี้
ที่อยู่ TextArea ไม่บังคับ
จังหวัด Select * จำเป็น
อำเภอ/เขต Select enable หลังเลือกจังหวัด
ตำบล/แขวง Select enable หลังเลือกอำเภอ
รหัสไปรษณีย์ Input (ReadOnly) ✔ เติมอัตโนมัติ
🔘 ปุ่มบนหน้าจอ

💾 บันทึก  ส่งข้อมูล ตรวจสอบ Validation แล้วบันทึก

🗑 ล้างฟอร์ม  ล้างข้อมูลทั้งหมดในฟอร์มกลับเป็นค่าว่าง

💡
ระบบจะแสดง Spin (โหลด) บน Dropdown จังหวัดขณะดึงข้อมูล ให้รอจนโหลดเสร็จก่อนเลือก
2

ค้นหาลูกค้า

/search-customer
📸 หน้าจอ – แบบฟอร์มค้นหา
รูปที่ 2 – หน้าค้นหา พร้อมตารางผลลัพธ์ (โหลดอัตโนมัติเมื่อเข้าหน้า)
📸 หน้าจอ – ผลลัพธ์การค้นหา
รูปที่ 3 – ตารางแสดงผลลัพธ์ลูกค้าทั้งหมด พร้อมปุ่มจัดการแต่ละแถว
📋 ขั้นตอนการใช้งาน
1
เข้าหน้าค้นหา

ระบบจะโหลดรายชื่อลูกค้าทั้งหมดมาแสดงในตารางโดยอัตโนมัติ

2
ระบุเงื่อนไขการค้นหา (ไม่บังคับ)

สามารถกรองได้ทีละหลายเงื่อนไขพร้อมกัน: เลขบัตร, ชื่อ, นามสกุล, จังหวัด, อำเภอ/เขต, ตำบล/แขวง, รหัสไปรษณีย์

3
กด ค้นหา

ตารางจะอัปเดตผลลัพธ์ตามเงื่อนไขที่กรอก

4
เลื่อน Pagination

หากผลลัพธ์มีหลายหน้า ใช้ปุ่ม ‹ / › เพื่อเปลี่ยนหน้า

5
กด ล้าง เพื่อรีเซ็ต

ล้างเงื่อนไขทั้งหมดและโหลดข้อมูลใหม่

🔎 เงื่อนไขการกรอง
ช่องรูปแบบ
เลขบัตรประจำตัวค้นแบบ Exact / Partial
ชื่อค้นแบบ Partial
นามสกุลค้นแบบ Partial
จังหวัดSelect Dropdown
อำเภอ/เขตSelect (enable หลังเลือกจังหวัด)
ตำบล/แขวงSelect (enable หลังเลือกอำเภอ)
รหัสไปรษณีย์Input (disable เมื่อเลือกตำบลแล้ว)

⚠️
รหัสไปรษณีย์จะถูก disable อัตโนมัติเมื่อเลือกตำบล/แขวงแล้ว เพื่อป้องกันข้อมูลขัดแย้งกัน
3

ดูรายละเอียดลูกค้า (Drawer)

/search-customer → ดูรายละเอียด
📸 หน้าจอ – Drawer รายละเอียด
รูปที่ 4 – Drawer สรุปข้อมูลลูกค้าแสดงจากทางขวา ไม่ออกจากหน้าค้นหา
📋 ขั้นตอนการใช้งาน
1
กด ดูรายละเอียด ในคอลัมน์จัดการ

ปุ่มอยู่ด้านขวาสุดของแต่ละแถวในตาราง

2
Drawer สไลด์จากขวา

แสดงแถบ Loading (Spin) ขณะดึงข้อมูล จากนั้นแสดงข้อมูลในรูปแบบ Descriptions

3
ปิด Drawer

กดปุ่ม ✕ บนมุมขวาบนของ Drawer หรือคลิกพื้นที่นอก Drawer

📄 ข้อมูลที่แสดงใน Drawer
#หัวข้อ
1เลขบัตรประจำตัว
2ชื่อ
3นามสกุล
4วันเกิด (dd/MM/yyyy)
5ที่อยู่
6ตำบล/แขวง
7อำเภอ/เขต
8จังหวัด
9รหัสไปรษณีย์
💡
Drawer ไม่นำทางออกจากหน้าค้นหา ทำให้สามารถดูข้อมูลหลาย record ได้โดยไม่ต้องกลับไปค้นหาใหม่
4

แก้ไขข้อมูลลูกค้า

/edit-customer/{id}
📸 หน้าจอ – แก้ไขข้อมูลลูกค้า
รูปที่ 5 – ฟอร์มแก้ไขข้อมูลลูกค้า โหลดข้อมูลเดิมมาให้อัตโนมัติ
📋 ขั้นตอนการใช้งาน
1
กด แก้ไข จากตารางค้นหา

ระบบจะนำทางไปยังหน้า /edit-customer/{id} พร้อมโหลด Spin ก่อน

2
ฟอร์มแสดงข้อมูลเดิมครบ

ทุกฟิลด์จะถูกเติมด้วยข้อมูลที่บันทึกไว้

3
แก้ไขข้อมูลที่ต้องการ

แก้ไขได้ทุกฟิลด์ (ยกเว้นรหัสไปรษณีย์ที่เป็น ReadOnly)

4
กด บันทึกการแก้ไข

ระบบตรวจสอบ Validation และบันทึกข้อมูลที่แก้ไข

5
หรือกด ยกเลิก

กลับไปหน้าก่อนหน้าโดยไม่บันทึกการเปลี่ยนแปลง

⚠️ กรณีไม่พบข้อมูล (Error State)
หาก ID ที่ระบุใน URL ไม่ตรงกับข้อมูลในระบบ จะแสดง Alert สีแดง พร้อมปุ่ม กลับ ให้กดกลับไปหน้าก่อนหน้าได้ทันที
🔘 ปุ่มบนหน้าจอ

💾 บันทึกการแก้ไข  ส่งข้อมูลที่แก้ไขแล้วบันทึก

↩ ยกเลิก  กลับหน้าก่อนหน้าโดยไม่บันทึก

✅
ฟอร์มแก้ไขใช้โครงสร้างเดียวกับฟอร์มเพิ่มลูกค้า ทำให้ใช้งานได้คุ้นเคย
✦

เคล็ดลับการใช้งาน

💡 ที่อยู่อัตโนมัติ
1
เลือกจังหวัดก่อนเสมอ

Dropdown อำเภอ/เขต จะ enable หลังเลือกจังหวัด

2
เลือกอำเภอ/เขต

Dropdown ตำบล/แขวง จะ enable ให้เลือกต่อ

3
เลือกตำบล/แขวง → รหัสไปรษณีย์กรอกให้เอง

ไม่ต้องพิมพ์รหัสไปรษณีย์เอง ระบบเติมให้อัตโนมัติ

🔍 เทคนิคการค้นหาให้เร็ว
✦
ค้นหาด้วยชื่อบางส่วน

ไม่จำเป็นต้องกรอกชื่อเต็ม เช่น "ทดสอบ" จะพบทุก record ที่มีคำนี้

✦
กรองด้วยจังหวัดเดียว

เลือกแค่จังหวัดโดยไม่ต้องเลือกอำเภอ/ตำบล ก็ได้ผลลัพธ์รายจังหวัด

✦
กด "ล้าง" เพื่อรีเซ็ตทันที

ไม่ต้องลบทีละช่อง กด "ล้าง" ครั้งเดียวเพื่อล้างทุกเงื่อนไข

🏷️ ตำนานสี & สัญลักษณ์
* สีแดง = ฟิลด์บังคับ (Required) สีเขียว / ตัวเอียง = ระบบเติมอัตโนมัติ สีส้ม / ReadOnly = อ่านได้อย่างเดียว แก้ไขไม่ได้ สีเทา = ไม่บังคับ (Optional)
